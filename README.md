# Solo_Project

Abstract:
Building a tool to list real estate properties digitaly, along with abilities to bookmark properties, search and filter and eventually reaching out to the agent to schedule a seeing for the estate.

Technologies used:
- Django
- Ajax
- SQL
- Bootstrap
